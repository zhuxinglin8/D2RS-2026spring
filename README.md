# 矿物-酶互作驱动冻土土壤有机碳累积与稳定 项目仓库说明

## 一、项目简介
本项目依托发表于《Environmental Science & Technology》的研究论文，围绕**青藏高原高海拔冻土**与**阿拉斯加高纬度北极苔原冻土**两大典型冻土区，系统开展土壤矿物、氧化酶活性与土壤有机碳稳定性的关联研究。

本仓库完整收纳本次研究的**野外采样数据、室内实验测定数据、数据处理代码、统计分析脚本、绘图源码、成果图件及全部补充资料**，可完整复现论文中所有数据处理、统计检验、机制分析与图表绘制全过程，可供后续科研学习、二次分析及模型拓展使用。

## 二、论文基础信息
- 论文英文题目：Mineral−Enzyme Interactions Drive Soil Organic Carbon Accumulation and Stabilization in Permafrost
- 期刊名称：Environmental Science & Technology（EST）
- 发表年份：2025
- 文献DOI：10.1021/acs.est.5c08036
- 研究核心：揭示矿物—过氧化物酶互作对冻土有机碳累积、保护及稳定的调控机制，对比高低海拔冻土碳稳定途径差异。

### 各文件夹功能说明
1. **data/**
存放野外采样信息、土壤分层数据、土壤理化性质、微生物残体碳、氨基糖、过氧化物酶活性、过氧化氢、活性矿物、碳同位素等原始与整理后数据集。

2. **code/**
包含数据清洗、异常值处理、相关性分析、统计建模、差异检验、机制解析、可视化绘图等全套可运行代码。

3. **figures/**
存放论文正式插图、机制概念图、分区对比图、相关分析图、补充分析附图等全部成果图件。

4. **supporting-info/**
存放研究补充实验方法、参数说明、原始测定记录表、辅助表格、文献支撑材料等。

## 四、研究区域与实验概况
### 1. 研究区域
- 青藏高原高寒冻土区：设置15个采样点位，分0–20 cm、20–40 cm两层土壤采样
- 阿拉斯加北极苔原冻土区：设置7个采样点位，分0–15 cm、15–30 cm两层土壤采样

### 2. 核心测定指标
- 微生物残体碳、氨基糖生物标志物
- 土壤过氧化物酶活性、过氧化氢浓度
- 短程有序活性矿物、CBD提取态铁氧化物
- 土壤有机碳组分、δ¹³C、Δ¹⁴C 碳同位素特征

## 六、代码运行说明
1. 本地下载整个项目文件夹，保持原有目录结构不变；
2. 使用RStudio打开code文件夹内脚本；
3. 运行顺序：**数据预处理 → 统计分析 → 相关性与机制分析 → 绘图出图**；
4. 代码运行结果与绘制图件将自动输出并保存至figures目录；
5. 所有文件路径已相对配置，无需额外修改路径即可直接运行。

## 八、项目研究意义简述
本项目区分了青藏高原**矿物主导型**与阿拉斯加**真菌-过氧化物酶共主导型**两套冻土碳稳定途径，阐明了过氧化物酶—过氧化氢—活性矿物的协同稳定机制，可为气候变暖背景下冻土碳流失风险评估、碳-气候反馈预测提供理论与数据支撑。
